Git & GitHub Cheatsheet

$ Git + GitHub workflow commands — বাংলায়

⚙ Setup
git config --global user.name "তোমার নাম"
Git username set করো
git config --global user.email "email@gmail.com"
Git email set করো
git config --list
সব configuration দেখো
💡 এটা একবার করলেই সব project এ কাজ করবে
🌱 Repo Start
git init
নতুন Git repository তৈরি করো
git clone https://github.com/user/repo
GitHub repo copy করো
git status
কি change হয়েছে দেখো
git log --oneline
সব commit history দেখো
📦 Add & Commit
git add .
সব files stage করো
git add filename.py
একটা file stage করো
git commit -m "fix navbar"
changes save করো
git commit --amend
শেষ commit edit করো
☁ Push & Pull
git push
GitHub এ upload করো
git pull
latest changes নামাও
git fetch
changes check করো
git push -u origin main
প্রথম push এর সময়
🌿 Branching
git branch
সব branch দেখো
git checkout new-feature
branch change করো
git checkout -b new-feature
নতুন branch তৈরি + switch
git merge new-feature
branch merge করো
↩ Undo / Fix
git restore .
সব unstaged changes remove
git reset HEAD~1
শেষ commit undo
git stash
temporary save
git stash pop
stash ফিরিয়ে আনো
🔗 GitHub Connect
git remote add origin https://github.com/user/repo.git
GitHub repo connect করো
git remote -v
connected remote দেখো
git branch -M main
main branch rename করো
git push -u origin main
GitHub এ প্রথম push
🐙 GitHub CLI
gh auth login
GitHub login
gh repo create
নতুন repo তৈরি
gh repo clone user/repo
repo clone
gh repo view
repo details দেখো
🔀 Pull Requests
gh pr create
নতুন Pull Request তৈরি
gh pr list
সব PR দেখো
gh pr checkout 12
PR locally checkout করো
💡 Pull Request team collaboration এর সবচেয়ে গুরুত্বপূর্ণ অংশ
