# 💻 Developer Cheatsheets Collection

A modern collection of interactive HTML cheatsheets for developers, covering **Git, GitHub, and Claude Code CLI**.

This project is built using pure HTML & CSS with a clean, responsive UI designed for fast learning and quick reference.

---

## 📦 What's Included

### 🌿 Git & GitHub Cheatsheet
A complete version control guide covering:

- Git setup & configuration
- Repository initialization
- Commit workflow
- Branching & merging
- Push / pull operations
- GitHub integration
- Pull requests
- Undo & recovery commands

---

### ⚡ Claude Code Cheatsheet
A CLI reference for Claude Code including:

- Session management
- Terminal navigation
- Slash commands
- Authentication
- Advanced usage
- Workflow tips
- Productivity shortcuts

---
