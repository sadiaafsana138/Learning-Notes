⚡ Quick Reference

Claude Code Cheatsheet

$ every command you need — all in one place

🚀
Start & Launch
claude
নতুন session শুরু করো (current folder এ)
claude --continue
আগের সর্বশেষ session continue করো
claude -c
--continue এর shortcut
claude --resume
পুরনো conversations এর list দেখাবে, বেছে নাও
📁
Folder Navigation (CMD)
cd FolderName
কোনো folder এ ঢোকো
cd ..
এক ধাপ পিছনে যাও
cd C:\Users\User\Documents
সরাসরি full path দিয়ে যাও
dir
বর্তমান folder এর সব files দেখো
⚡
Quick / One-off Commands
claude "তোমার প্রশ্ন এখানে"
session না খুলেই সরাসরি প্রশ্ন করো
claude -p "fix bug in app.py"
--print shortcut, output terminal এ দেখাবে
claude --version
installed version চেক করো
claude doctor
installation ঠিকঠাক আছে কিনা চেক করো
💬
Inside Session — Slash Commands
/help
সব command দেখো
/exit
session বন্ধ করো
/clear
chat history মুছো
/status
account info দেখো
/model
AI model বদলাও
/cost
কত টাকা খরচ হলো
🔐
Login & Auth
claude
প্রথমবার চালালে browser এ login হবে automatically
claude logout
logout করো
claude login
আবার login করো
⚠ দরকার
Claude Pro / Max / Team account ছাড়া কাজ করবে না
💡
Pro Tips
TIP
সবসময় project folder এ গিয়ে claude চালাও, তাহলে Claude তোমার files দেখতে পাবে।
TIP
Install এর পর CMD বন্ধ করে নতুন করে খোলো — না হলে claude command কাজ নাও করতে পারে।
TIP
Windows Terminal app ব্যবহার করলে output সুন্দর দেখায়। Microsoft Store থেকে free নামাও।
NOTE
কোনো সমস্যা হলে claude doctor চালাও — বেশিরভাগ সমস্যা ধরে দেবে।
🔧
Advanced Usage
claude -p "review app.py" > result.txt
Claude এর output একটা file এ save করো
claude --model claude-opus-4-5
নির্দিষ্ট model দিয়ে চালাও
claude --help
সব available flags ও options দেখো
exit
CMD window বন্ধ করো

Daily Workflow
01
CMD / PowerShell খোলো
02
cd তোমার-project-folder
project folder এ যাও
03
claude -c

✓
